Extracts text from a DOC format Microsoft Word document. Uses the catdoc command-line utility to do so. Catdoc can be downloaded for Linux at:

http://wagner.pp.ru/~vitus/software/catdoc/

The path to the catdoc binary will be detected automatically if it lives at /usr/bin/catdoc or /usr/local/bin/catdoc. If your catdoc binary is in a non-standard place, you can set it in your _ss_environment.php file like so:

  1.  define('CATDOC_BINARY_LOCATION', '/home/username/bin/catdoc');

Or, if using _config.php, you can also set it directly on the class:

  1.  CatdocDocExtractor::$binary_location = '/home/username/bin/catdoc';
